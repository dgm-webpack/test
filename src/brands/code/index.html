Q&A

產品味道如何？


淡淡水蜜桃清香風味大人小孩都喜歡喔！

白蘭氏木寡醣乳酸菌怎麼吃？


餐後食用。可直接倒進嘴裡，或加入冷開水中食用勿用熱水沖開。每袋開封後，請馬上食用完。

小孩可以吃嗎？


4~9歲每天食用1包，10歲以上每天食用1~2包。

素食者可以食用嗎？


產品中有優格香料，是乳製品喔!如果你是奶素，是沒有問題的！

產品需要放冰箱嗎？


只要將產品放在陰涼乾燥處，並避免光線直接曝曬在保存期限內，都可以安心食用，不需要放置冰箱貯藏。
